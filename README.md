# View Counter App

Приложение-счётчик для [Books App](https://github.com/tfomina/Books-App)

Обрабатывает два роута

1. увеличить счётчик

```
POST /counter/:bookId/incr
```

2. получить значение счётчика

```
GET /counter/:bookId
```

### Запуск программы в dev режиме

```javascript
npm run dev
```
